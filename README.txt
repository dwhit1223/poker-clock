------------------------------------------------------------
OVERVIEW
------------------------------------------------------------

Poker Clock Pro is a fullscreen tournament clock designed
for home games and small poker clubs.

It provides clear blind progression, break management,
prize pool tools, and customizable branding in a clean,
TV-friendly interface.

The application runs locally for maximum reliability.


------------------------------------------------------------
SYSTEM REQUIREMENTS
------------------------------------------------------------

• Windows 10 or Windows 11 (64-bit)
• Large display recommended (TV or projector)
• No internet connection required after activation


------------------------------------------------------------
GETTING STARTED
------------------------------------------------------------

1. Extract the downloaded ZIP file.
2. Open the extracted folder.
3. Double-click the PokerClock application file.
4. Click "Fullscreen" for best results on a TV.

No installation is required.
The application runs locally from the folder.


------------------------------------------------------------
FEATURES
------------------------------------------------------------

Core Features (Free Version)

• Blind level timer
• Break rounds
• Buy-in and rebuy tracking
• Prize pool calculations
• Multiple blind structure templates
• Large-screen optimized layout


Pro Features

• Custom logo upload
• Theme customization (fonts and colors)
• Custom sound uploads
• Save and load tournament configurations
• Enhanced branding options


------------------------------------------------------------
SOUND NOTES
------------------------------------------------------------

The application includes default sound effects for:

• Blind level increases
• One minute remaining
• Break start

Pro users may upload custom audio files (MP3 recommended).


------------------------------------------------------------
CONFIGURATION FILES (PRO)
------------------------------------------------------------

Saved configurations are exported as JSON files.

These files include:

• Blind structure
• Prize distribution
• Branding settings
• Theme customization
• Custom sounds (if used)

You can load these files to quickly restore tournament setups.


------------------------------------------------------------
BEST PRACTICES
------------------------------------------------------------

• Use fullscreen mode on a TV
• Keep sound levels moderate
• Use high-contrast colors for visibility
• Test sounds before starting the tournament


------------------------------------------------------------
TROUBLESHOOTING
------------------------------------------------------------

If the app does not launch:

• Ensure you extracted the ZIP file fully
• Confirm you are running Windows 10 or newer
• Try running as administrator if necessary

If audio does not play:

• Click anywhere in the app once (unlocks audio)
• Confirm system volume is enabled

------------------------------------------------------------
UPDATES
------------------------------------------------------------

To update Poker Clock Pro:

1. Download the newest version from Gumroad
2. Extract the ZIP
3. Replace your previous PokerClock folder
